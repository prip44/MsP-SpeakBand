One-Time Setup Required

Connect this app to Firebase so student data syncs across devices

Create a Firebase Project

a
Go to console.firebase.google.com and sign in with a Google account
b
Click "Create a project" → name it "SpeakBand" → click Continue → Create Project
c
Click the web icon </> to register your app → name it anything → click "Register app"

Enable Realtime Database

a
In the left sidebar, click "Build" → "Realtime Database"
b
Click "Create Database" → choose any location → select "Start in test mode" → click Enable

Copy Your Config

a
Click the gear icon ⚙️ top-left → "Project settings"
b
Scroll to "Your apps" → you'll see a firebaseConfig block like this:
const firebaseConfig = { apiKey: "AIzaSyBxxxxxxxxxxxxxxxxxxxxxxx", authDomain: "speakband-xxxxx.firebaseapp.com", databaseURL: "https://speakband-xxxxx-default-rtdb.firebaseio.com", projectId: "speakband-xxxxx", storageBucket: "speakband-xxxxx.appspot.com", messagingSenderId: "123456789", appId: "1:123456789:web:abcdef123456" };

Paste Config Into This File

a
Open this HTML file in any text editor (Notepad, VS Code, etc.)
b
Search for PASTE YOUR FIREBASE CONFIG HERE
c
Replace the entire firebaseConfig block with your own values from step 3. Save the file.

Host It (GitHub Pages — Free)

a
Go to github.com → create free account
b
Click "New repository" → name it speakband → Create
c
Click "Add file" → "Upload files" → upload this HTML as index.html
d
Go to Settings → Pages → Branch: main → Save
e
Wait 1 minute. Your site is live at https://YOUR_USERNAME.github.io/speakband/

After setup: Refresh this page. If you see the SpeakBand landing page with a green dot, it worked!
